Hermes Agent System Doctor is maintained by Aleksei Ulianov / Sprut_AI.

Canonical source: https://github.com/AlekseiUL/hermes-system-doctor

This is an independent companion diagnostic tool for Hermes Agent installations. It is not an official Nous Research / Hermes Agent component unless explicitly accepted upstream.

The project is intended to be read-only by default: it inventories local Hermes configuration surfaces and produces redacted reports without mutating the installation.
